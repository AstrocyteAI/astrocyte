```python
from astrocytes import Astrocyte

brain = Astrocyte.from_config("astrocytes.yaml")

await brain.retain("Calvin prefers dark mode", bank_id="user-123")

hits = await brain.recall("What are Calvin's preferences?", bank_id="user-123")
```

<span class="section-label" aria-hidden="true">⟩ What it does</span>
<h2 class="section-heading">Three operations, any backend</h2>
<div class="section-subheading">
  Retain, recall, and reflect — with pipelines and policies that stay valid when you swap storage or engines.
</div>

<div class="feature-grid not-content feature-grid--cols-3">
  <div class="feature-card">
    <h3>Retain</h3>
    <p>Store memories with automatic chunking, entity extraction, and embedding. PII scanning and content validation happen before anything reaches your backend.</p>
  </div>
  <div class="feature-card">
    <h3>Recall</h3>
    <p>Multi-strategy retrieval — semantic, graph, keyword, temporal — fused with reciprocal rank fusion and reranked. Token budgets enforced automatically.</p>
  </div>
  <div class="feature-card">
    <h3>Reflect</h3>
    <p>Synthesize answers from memory. Native engine reflect or automatic fallback via LLM. Disposition-aware personality for support, coding, or research agents.</p>
  </div>
</div>

{/* ── Policy layer ─────────────────────────────────── */}

<span class="section-label" aria-hidden="true">⟩ Governance</span>
<h2 class="section-heading">Built-in governance</h2>
<div class="section-subheading">
  Neuroscience-inspired policies that protect every operation — regardless of backend.
</div>

<div class="feature-grid not-content feature-grid--cols-3">
  <div class="feature-card">
    <h3>PII barriers</h3>
    <p>Regex, NER, or LLM-based scanning. Redact, reject, or warn — before content reaches the provider.</p>
  </div>
  <div class="feature-card">
    <h3>Rate limits & budgets</h3>
    <p>Token bucket rate limiting, daily quotas, and per-operation token budgets. Prevent runaway agents.</p>
  </div>
  <div class="feature-card">
    <h3>Circuit breakers</h3>
    <p>Automatic degraded mode when backends go down. Empty recall, error, or cache — with controlled recovery.</p>
  </div>
  <div class="feature-card">
    <h3>Observability</h3>
    <p>OpenTelemetry spans and Prometheus metrics on every operation. Switch providers without rebuilding dashboards.</p>
  </div>
  <div class="feature-card">
    <h3>Access control</h3>
    <p>Per-bank read/write/forget/admin permissions. Principals, grants, and audit trails built in.</p>
  </div>
  <div class="feature-card">
    <h3>Data governance</h3>
    <p>Classification levels, compliance profiles (GDPR, HIPAA, PDPA), data residency, encryption, and DLP.</p>
  </div>
</div>

{/* ── Integrations ─────────────────────────────────── */}

<span class="section-label" aria-hidden="true">⟩ Integrations</span>
<h2 class="section-heading">Agents, frameworks, models</h2>
<div class="section-subheading">
  One framework, every agent stack.
</div>

<CardGrid>
  <Card title="MCP Server" icon="laptop">
    Any MCP-capable agent (Claude Code, Cursor, Windsurf) gets memory via `astrocytes-mcp`. Zero code integration.
  </Card>
  <Card title="Agent Frameworks" icon="puzzle">
    LangGraph, CrewAI, Pydantic AI, OpenAI Agents SDK, AG2 — thin middleware wires Astrocytes into each framework's memory abstraction.
  </Card>
  <Card title="LLM Providers" icon="translate">
    Multi-provider gateways (e.g. LiteLLM, Portkey, OpenRouter, Vercel AI Gateway) or direct SDKs — OpenAI, Anthropic, Bedrock, Azure, Vertex, Ollama — or bring your own via the LLM SPI.
  </Card>
  <Card title="Memory Portability" icon="right-arrow">
    Export and import memories between providers using Astrocytes Memory Archive (AMA), a portable newline-delimited JSON (JSONL) format. No vendor lock-in.
  </Card>
</CardGrid>
